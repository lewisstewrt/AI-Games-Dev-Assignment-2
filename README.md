# Assignment 2 – Voice Interaction Game

## Overview
This project is a 2D game built using C++ and SDL3.  
It includes voice interaction features where the player can speak to NPCs using speech-to-text and receive AI-generated responses.

---

## Features
- Player movement
- Item collection
- Inventory system
- NPC interaction
- Voice input (speech-to-text)
- AI responses
- Voice output (text-to-speech)
- Basic trading system

---

## Controls
- W / A / S / D – Move  
- Arrow Keys – Move  
- Space – Talk to NPCs (voice input)

---

## Gameplay
- Collect items  
- Interact with NPCs  
- Complete tasks by trading and progressing through the game  

---

## Files
- `src/` – Source code  
- `resources/` – Game assets  
- `CMakeLists.txt` – Build configuration  
- `assignment2-voice-interaction.pdf` – Assignment specification  

---

## Setup
Build using CMake and run the project in Visual Studio or your preferred IDE.
